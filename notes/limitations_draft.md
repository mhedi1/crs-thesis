# Limitations Draft

The proposed system is a research prototype and has several limitations. First, the KBRD baseline is based on legacy ParlAI code, which makes the integration difficult because of hardcoded paths, old dependencies, and strict CUDA requirements. Second, the Qwen3.5:35b model is accessed through an external university API, so evaluation speed depends on server availability and load. Third, the official evaluation uses strict normalized exact title matching, which is academically conservative but may undercount cases where the system recommends a correct movie under a slightly different title or alias. Finally, response generation metrics such as BLEU and ROUGE are limited for conversational recommendation because many different natural responses can be valid for the same dialogue.
